# Building with the Claude API

Exercises and experiments from Anthropic Academy's
[Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api) course.

## Overview

This repo covers core concepts for working with Anthropic models via the Claude API,
including tool use, prompt engineering, structured outputs, and agentic workflows.

## Topics Covered

- API setup and authentication
- Message structure and multi-turn conversations
- Tool use and JSON schema definitions
- Prompt evaluation methods
- Agentic loops and multi-tool workflows
- Batch processing and token optimization

## Stack

- Python
- `anthropic` SDK
- JSON (tool schemas)
